# 1. Introdução
  ## a. Tema
  Aplicativo móvel para adoção de animais domésticos.
  
  ## b. Objetivo do projeto
  Facilitar o contato entre quem quer adotar e os abrigos de animais, através da ONG da cidade.
  
  ## c. Justificativa do tema escolhido
  Aumentar a visibilidade de animais resgatados e simplificar o processo de comunicação entre o adotante e o abrigo.
  
  ## d. Método de trabalho
  Criado com React Native e Expo.
  
  ## e. Organização do trabalho
  O projeto foi organizado em pastas de telas (screens), componentes e lógica centralizada no arquivo principal App.js.
  

# 2. Descrição geral do Sistemas
  ## a. Descrição do problema

  ## b. Principais envolvidos e suas características
